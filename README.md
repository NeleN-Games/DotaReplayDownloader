## 🚀 Key Features
* **Cloud-Powered:** Uses GitHub Actions to process and parse replay data, bypassing regional network issues.
* **High-Speed:** Utilizes **Aria2c** with 16-thread support for maximum download speed.
* **Smart Caching:** Automatically checks for existing replays in the cloud to save time.
* **Easy Setup:** Simple interface and straightforward configuration.

---

### 📥 Getting Started

To get started, follow these simple steps:

#### 1. Setup (Fork & Token)
1. **Fork this repository** to your own GitHub account to enable the cloud processing pipeline.
2. **Generate a GitHub Token:** 
   - Go to [GitHub Settings](https://github.com/settings/tokens).
   - Create a new **Personal Access Token (classic)**.
   - Select the `repo` and `workflow` scopes.
   - Save your token securely.

#### 2. Download the App
Since releases are managed in the main repository, download the latest version of the executable here:
**[👉 Click here to download DRD.exe from the official Releases page](https://github.com/NeleN-Games/DotaReplayDownloader/releases)**

#### 3. Run and Configure
1. Extract the downloaded `.zip` file.
2. Run `DRD.exe`.
3. Enter your **GitHub Token**, **Username**, **Repository Name**, and your **Dota 2 installation path** when prompted.
4. Enter any **Match ID** to start the download!

---

### ⚠️ Important Notes
* **Replay Availability:** Remember that Valve typically keeps replay files available for about 8–10 days.
* **Actions Enabled:** Ensure that "GitHub Actions" are enabled in the settings of your forked repository.
